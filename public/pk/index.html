Как поставить Happ на Windows

Стандартная установка (x64)

Запускаешь скачанный exe. Windows может предупредить «Издатель не проверен» — нажми «Подробнее» → «Выполнить в любом случае». В мастере установки достаточно нажимать «Далее» — нестандартных опций там нет.

После установки приложение запускается автоматически. Иконка Happ — синий ромб с буквой H — появляется в трее (рядом с часами). Кликни по ней, чтобы открыть основное окно с профилями.

Portable-вариант

Распакуй ZIP-архив в любую папку (Документы, рабочий стол, флешка). Запусти happ.exe двойным кликом. Профили и настройки будут храниться в той же папке, рядом с экзешником.

Какие требования у Happ на ПК

Минимум — Windows 7 SP1 с обновлениями, 2 ГБ оперативной памяти и 100 МБ свободного места на диске. Процессор — любой 64-битный (для x64-версии). На совсем старых машинах без AVX-инструкций нужна отдельная сборка — она тоже доступна на странице загрузки.

Happ на ноутбуке

На ноутбуке всё работает идентично десктопу. Единственный нюанс — приложение поднимает защищённый туннель, и при подключении к новой сети (например, в кафе) рекомендуется отключать туннель и заново подключаться после авторизации в этой сети. Иначе captive portal не откроется.

Happ Plus на пк

Подписка Plus работает на ПК так же, как на телефоне: импортируешь профиль из бота, активируешь его и подключаешься. Один профиль покрывает компьютер и до двух дополнительных устройств одновременно.

Какие требования у Happ на пк?
Минимум — Windows 7 SP1, 2 ГБ ОЗУ, 100 МБ места. Процессор — любой 64-битный. Для старых CPU без AVX есть отдельная сборка.
Как настроить Happ на ПК?
После установки — открой Telegram-бот, скопируй ссылку-профиль, открой Happ, нажми «+» в углу → «Импорт из буфера». Профиль появится — тап по нему, потом по большой кнопке.
Happ на ноутбуке тормозит — что делать?
Чаще всего причина в фоновом антивирусе, который проверяет трафик туннеля. Добавь happ.exe в исключения антивируса и сетевого экрана.
Есть ли Happ Desktop сборка?
Да, обычный установщик и portable — это и есть Desktop-варианты. Под Linux отдельной сборки пока нет, но работает через Wine/Bottles.
Можно ли поставить Happ на Windows 7?
Да, есть специальная сборка для Windows 7 и процессоров без AVX. Она чуть тяжелее, но запускается даже на 14-летнем железе.
